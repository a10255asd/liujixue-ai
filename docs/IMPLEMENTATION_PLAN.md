## 停止规则

以下情况不要继续开发：

- 内容模型未更新却直接写页面。
- 面试题没有考察点和追问。
- 页面为了炫技牺牲可读性。
- 复制外部课程、书籍或网站长文。
- 尚未本地 build 通过就部署。
- 未明确域名/DNS 状态就宣称上线完成。
