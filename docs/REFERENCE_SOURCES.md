# 参考资料和内容来源原则

更新时间：2026-07-15

## 来源优先级

1. 官方文档。
2. 官方示例或可运行 demo。
3. 论文和技术报告。
4. 高质量工程博客。
5. 个人实践记录。

## 内容写作规则

- 引用官方定义时要注明来源。
- 不要长篇复制官方文档。
- 不要把第三方博客观点写成确定事实。
- 对“最佳实践”类内容要写清适用场景。
- 过时风险高的内容要标注更新时间。

## 面试题来源规则

可以来自：

- 官方文档中的核心概念。
- 岗位 JD 的技能要求。
- 自己项目实践中的真实问题。
- 面试复盘。
- 开源项目 issue 和文档。

不可以：

- 复制付费题库。
- 伪造大厂面试题来源。
- 不加判断地搬运网上答案。
